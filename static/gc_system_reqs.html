GoldCare Workstation System Requirements

CloudCare 2.0 or newer:

GoldCare's CloudCare 2.0 version is web browser based.   Connection to GoldCare is supported by any device running current releases of Google Chrome or Microsoft Edge web browsers.  For the best experience, we recommend using a device with a large display such as a workstation, laptop or tablet.  

Note that pop-ups must be enabled in the browser for accessing GoldCare.   For instructions on enabling pop-ups, consult the documentation for your selected browser.

CloudCare 17 or earlier:

The CloudCare17 or earlier versions of GoldCare require the GoldCare Client software installed and configured to connect to the server.  Below are minimum recommendations for an end user workstation.  

Workstation Hardware

  • Microsoft Windows Certified hardware
  • CPU: x86 compatible running 3.4 GHz recommended (3Ghz min.)
  • RAM: 8GB recommended
  • Screen size: 1920 x1080 recommended (1280 x 1024 min.)

Workstation Software

  • Operating system: Windows 10
  • Browser: Google Chrome or MS Edge
  • GoldCare Client Software

Note that for all versions of GoldCare Microsoft Excel must be installed on the device to use the Make Your Own Reports export to Excel feature.
